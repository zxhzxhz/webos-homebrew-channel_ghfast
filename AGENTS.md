# AGENTS.md

## Purpose

Mirror of [webos-homebrew-channel](https://github.com/webosbrew/webos-homebrew-channel)'s app catalog (`apps.json`) with GitHub download URLs rewritten through `ghfast.top` for accelerated access in regions with slow GitHub connectivity.

## Source

- **Upstream JSON**: `https://repo.webosbrew.org/api/apps.json`
- **Acceleration proxy**: `https://ghfast.top/` (prefixes GitHub URLs)

## URL rewriting rules

Every `https://github.com/` and `https://raw.githubusercontent.com/` URL in `apps.json` is prefixed with `https://ghfast.top/`. Example:

```
Before: https://github.com/webosbrew/youtube-webos/releases/download/v0.5.3/youtube.leanback.v4_0.5.3_all.ipk
After:  https://ghfast.top/https://github.com/webosbrew/youtube-webos/releases/download/v0.5.3/youtube.leanback.v4_0.5.3_all.ipk
```

Non-GitHub URLs (`repo.webosbrew.org`, `mirrors.kodi.tv`, relative paths like `/api/apps/...`) are left untouched.

## GitHub Actions

- **Workflow**: `.github/workflows/sync.yml`
- **Schedule**: hourly (`cron: 0 * * * *`) + manual `workflow_dispatch`
- **Behavior**: fetch fresh `apps.json` from upstream every run, apply sed rewrites, commit + push if changed

## Local testing

```sh
curl -fsSL -o apps.json https://repo.webosbrew.org/api/apps.json
sed -i 's|https://github\.com/|https://ghfast.top/https://github.com/|g' apps.json
sed -i 's|https://raw\.githubusercontent\.com/|https://ghfast.top/https://raw.githubusercontent.com/|g' apps.json
```
